# Tech Stack Manual

## 1. System Overview
This is a Learning Platform developed by Nashville Software School (NSS). The system implemented Github SSO and let you in through GitHub login.

It has several components to manage the learning cohorts and students. 

Tabs:
Overview:
Displays list of cohorts. Seperate views for active and past cohorts.
Inside the cohort there are student enrollment details, option to tranfer student, attendance tracker, course details, material links.,

Capstones:
Contains list of capstone projects which might be used at the end of course as assessment to complete and pass the course.

Coreskills:
Shows the set of skills whcih helps for efficient and effective learning.

### 3b. How to Start It
1. make setup - command to setup everything and run the project initially.
2. make up - to spin up all the Docker containers(db,api,client), create network for containers to talk and make system running later after the initial setup.
3. make restart - stops and removes everything related to container and again builds the container. If container don't spin up correctly, we can use this option.
4. make up-api - spins up only api container.
5. make up-client - spins up only react client container.
